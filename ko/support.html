Fridoumi는 AI가 냉장고 속 재료를 기반으로 레시피를 추천하는 요리 도우미 앱입니다. 재료를 등록하면 만들 수 있는 음식을 자동으로 제안하고, 유통기한 알림으로 음식 낭비를 줄여드립니다.
냉장고에 등록된 재료를 분석해 AI가 최적의 레시피를 자동으로 추천합니다. 무료 사용자는 하루 제한된 횟수만 추천을 받을 수 있으며, Pro 구독자는 무제한으로 이용 가능합니다.
영수증 스캐너 화면에서 마트 영수증 사진을 선택하거나 촬영하면, AI가 재료를 자동으로 인식하여 팬트리에 추가합니다. 영수증 스캔은 Pro 구독자 전용 기능입니다.
영수증이 선명하게 찍혀있는지 확인해주세요. 밝은 조명 아래에서 촬영하면 인식률이 크게 향상됩니다. 영수증이 구겨지거나 흐릿하면 인식이 어려울 수 있습니다. 계속 문제가 발생하면 yoonseok84@gmail.com으로 문의해주세요.
iPhone 설정 → Apple ID → 구독 → Fridoumi → 구독 취소를 선택하시면 됩니다. 해지 후에도 현재 구독 기간이 끝날 때까지 Pro 기능을 이용하실 수 있습니다. Fridoumi는 직접 구독을 처리하지 않으며, 모든 구독은 Apple을 통해 관리됩니다.
Pro에는 무제한 AI 레시피 추천, 영수증 스캔, 스마트 장보기, 광고 제거가 포함됩니다. iCloud 동기화는 모든 사용자에게 무료로 제공됩니다. 월간(₩5,900/월) 또는 연간(₩44,000/년, 33% 할인) 중 선택하실 수 있습니다.
모든 팬트리 데이터와 레시피는 기기 내 SwiftData에 로컬로 저장됩니다. 모든 사용자는 무료로 iCloud 동기화를 선택적으로 활성화할 수 있으며, iCloud 데이터는 Apple이 암호화하여 관리합니다. Fridoumi는 자체 서버를 운영하지 않습니다.
Apple의 표준 환불 정책을 따릅니다. 환불을 요청하려면 reportaproblem.apple.com에서 Apple ID로 로그인하여 신청하세요.
앱 첫 실행 시 알림 권한을 허용하시면, 재료 유통기한 3일 전에 자동으로 알림이 전송됩니다. 알림 설정은 iPhone 설정 → 알림 → Fridoumi에서 변경할 수 있습니다.
앱 삭제 시 기기에 저장된 모든 로컬 데이터가 영구적으로 삭제됩니다. iCloud에 동기화된 데이터(Pro 구독자)는 별도로 관리됩니다. iCloud 데이터를 삭제하려면 iPhone 설정 → Apple ID → iCloud → 저장 공간 관리 → Fridoumi → 데이터 삭제를 선택하세요.
재료 데이터는 레시피 추천 생성을 위해 AI 서비스 제공자에게만 전송됩니다. 영수증 이미지(Pro)는 분석 후 즉시 폐기됩니다. 데이터를 판매하지 않습니다. 자세한 내용은 개인정보처리방침을 확인해 주세요.
