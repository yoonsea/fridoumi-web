1. 서비스 개요

Fridoumi("본 앱")는 YoonLab이 개발한 AI 기반 냉장고 관리 및 레시피 추천 애플리케이션입니다. Fridoumi를 다운로드하거나 사용함으로써 본 이용약관에 동의하는 것으로 간주됩니다. 동의하지 않으시는 경우 앱을 사용하지 마십시오.

2. 구독 서비스

요금

  • Pro 월간 — ₩5,900 / 월
  • Pro 연간 — ₩44,000 / 년 (33% 할인)

자동 갱신

구독은 각 결제 기간 종료 시 자동으로 갱신됩니다. 갱신일 24시간 전까지 해지하지 않으면 자동으로 결제됩니다. 구매 확인 시 등록된 Apple ID로 요금이 청구됩니다.

해지 방법

iPhone 설정 → Apple ID → 구독 → Fridoumi → 구독 취소를 선택하시면 됩니다. 해지는 현재 구독 기간 종료 시점에 적용되며, 그때까지 Pro 기능을 계속 이용하실 수 있습니다.

환불

Apple의 환불 정책에 따릅니다. 환불을 요청하려면 reportaproblem.apple.com에서 신청하세요.
